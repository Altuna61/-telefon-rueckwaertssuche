# Anforderungsanalyse – Telefonverzeichnis-Tool

## 1. Funktionale Anforderungen

- **F1:** Das System erhält eine Telefonnummer als Eingabe (Konsole).
- **F2:** Das System stellt eine Anfrage an das externe Verzeichnis „Das Örtliche“.
- **F3:** Das System muss aus den Ergebnissen Name und ggf. Adresse extrahieren.
- **F4:** Das System gibt das gefundene Ergebnis lesbar aus.
- **F5:** Keine Ergebnisse müssen entsprechend ausgegeben werden.

## 2. Nicht-funktionale Anforderungen

- **NF1:** Nutzeroberfläche soll verständlich sein (Konsole).

## 3. Randbedingungen

- **RB1:** Python 3.x wird verwendet.
- **RB2:** Bibliotheken: `requests` + `BeautifulSoup4`.
